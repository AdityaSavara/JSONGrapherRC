The main example is intended to be like a tutorial:
exampleUsageJSONRecordCreator.py

The other subdirectories are additional example types.

example2 shows how to launch a JSONGrapher window for dragging and dropping in multiple files for merging and plotting.

example3 shows an example where the json record calls JavaScript source code from online to simulate and populate a dataseries.

example4 shows an example where custom units are used.
